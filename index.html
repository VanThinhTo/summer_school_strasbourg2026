Track 01

Orange-Chemoinformatics

1

Work around Orange as a no-code, visual environment for cheminformatics analysis, with documentation, tutorials, and customized widgets for molecular workflows.

Orange visual workflows widgets tutorials
  • Documentation and tutorial material for the Orange add-on.
  • New-widget concept covering SMILES input, ETKDG, OpenMM MD, and 3D SDF output.
  • Conformer generation settings with ranked outputs.

Track 03

Conformal Prediction in QSPRpred

3

Extensions and fixes for QSPRpred, an open-source modelling toolkit covering data preparation, model training, validation, visualization, and deployment.

QSPR/QSAR PCA UMAP t-SNE
  • CLI bug fix plus tests for new plotting functionality.
  • PCAPlot, UMAPPlot, and TSNEPlot utilities for chemical-space visualization.
  • Biased splitting methods and tests, including activity-cliff and kNN-failure scenarios.
Code Slides

Track 04

MMPA Launchpad

4

Agentic cheminformatics tool for matched molecular pair analysis, extending ChemSpace Copilot with MMPDB-backed transformations, predictions, and generated analogues.

MMPA mmpdb agents ChemSpace Copilot
  • Integration concept for an MMPA toolkit and prompts inside the agent workflow.
  • Pipeline covering fragment, index, loadprops, transform, predict, and generate.
  • CSV artifacts for transform results, predicted property changes, and generated structures.

What Was Built

Hackathon Outputs

The work concentrates on practical tooling: better interfaces, clearer tutorials, tested modelling utilities, and agent-ready cheminformatics workflows.

Workflow Interfaces

Orange-Chemoinformatics focused on visual analysis and widget workflows for molecular data tasks.

Model Diagnostics

QSPRpred gained dimensionality-reduction plots, biased split methods, CLI fixes, and supporting tests.

Agentic MMPA

MMPA Launchpad connected matched-pair database operations to an agent workflow for transformations and analogue generation.

Reaction Exploration

The rIGM track points toward quantum-level reaction-path exploration materials shared with the group.
